##### алгоритм создания простого проекта

Главный файл это index.html.  В него в <div id="app"></div> и всталяетсчя весь контент
Файл src/main.ts в приложении Vue.js является точкой входа для вашего приложения. Он отвечает за инициализацию Vue-приложения, настройку глобальных параметров и монтирование корневого компонента в DOM.

#### установка дополонительных пакетов

npm install ant-design-vue установка ant-design-vue
ant-design-vue это библиотека компонентов пользовательского интерфейса для Vue.js, основанная на дизайне Ant Design, который был разработан компанией Alibaba. Она предоставляет набор высококачественных, готовых к использованию компонентов, которые помогают разработчикам быстро создавать красивые и функциональные веб-приложения. 
Аналог bootstrap 5

npm list ant-design-vue установлен ли пакет
